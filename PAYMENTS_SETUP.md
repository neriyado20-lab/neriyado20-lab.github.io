# הפעלת תשלומים באתר גל עיני

כל עוד `enabled` מוגדר `false` בקובץ `payments-config.js`, לא ניתן לבצע חיוב.

## מה לקבל מחברת הסליקה

- קישור תשלום חודשי מתחדש.
- קישור תשלום שנתי מתחדש.
- כתובת לקבלת הודעות Webhook על תשלום, ביטול וכשל חיוב.
- פרטי מסוף ומפתחות API לשמירה בשרת בלבד.

## הפעלה ראשונית

1. להזין את שני קישורי התשלום בשדות `paymentUrl`.
2. להזין כתובת תמיכה.
3. לשנות `enabled` ל-`true`.
4. לבדוק עסקה בסביבת בדיקות לפני חיוב אמיתי.

## אבטחה

אין להכניס לקובצי GitHub Pages מפתח API, סיסמה, מספר חשבון בנק או סוד Webhook.
הפעלת הרשאות אוטומטית למשתמשים דורשת שרת נפרד שמאמת Webhooks ושומר מנויים.
