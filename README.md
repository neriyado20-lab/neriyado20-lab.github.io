# אתר גל עיני - גרסה חינמית ללא AI

תיקייה זו היא אתר סטטי שאפשר לפתוח ישירות בדפדפן או להעלות בחינם ל-GitHub Pages.

## קבצים

- `index.html` - עמוד הבית.
- `web.html` - עבודה באתר עם מנוע החיפוש.
- `examples.html` - עמוד צפנים לדוגמה.
- `ai.html` - כלי עזר מקומי לבניית רשימת מילים לעיון.
- `order.html` - הכנת בקשת חיפוש לפי הזמנה.
- `contact.html` - פניות צור קשר הנשמרות בדפדפן המקומי.
- `admin.html` - לוח ניהול מקומי לנתונים שנשמרו בדפדפן הזה.
- `admin-config.js` - הגדרות כניסה והעלאה לניהול.
- `styles.css` - עיצוב האתר.
- `downloads/gal_einai_V392_with_AI.zip` - קובץ ההפצה העדכני עם הפניה לחלון התשלום כאשר פענוח AI חי דורש קרדיט, עד דילוג 5000.
- `downloads/gal_einai_V392_no_AI.zip` - קובץ ההפצה העדכני ללא AI, עד דילוג 5000.
- `downloads/gal_einai_V392_with_AI_setup.exe` - מתקין Windows מלא ונוח עם AI, קובץ אחד להפעלה.
- `downloads/gal_einai_V392_no_AI_setup.exe` - מתקין Windows מלא ונוח בלי AI, קובץ אחד להפעלה.
- `downloads/gal_einai_V392_with_AI_patch.zip` - עדכון קטן עם AI למחשב שכבר הותקנה בו החבילה המלאה.
- `downloads/gal_einai_V392_no_AI_patch.zip` - עדכון קטן בלי AI למחשב שכבר הותקנה בו החבילה המלאה.
- `downloads/` - נשמרות חמש גרסאות אחרונות נגישות.
- `downloads/archive/` - גרסאות ישנות יותר נשמרות לארכיון.
- `archive/` - קבצים ישנים או לא מקושרים שנשמרו לארכיון במקום להימחק.

## הפעלה מקומית

פותחים את `index.html` בדפדפן.

## התקנת תוכנת Windows למכותבים

למשתמש חדש מומלץ לשלוח את אחד מקובצי ההתקנה:

- `gal_einai_V392_with_AI_setup.exe`
- `gal_einai_V392_no_AI_setup.exe`

המשתמש מפעיל את קובץ ההתקנה, עובר את אשף ההתקנה, ומקבל קיצור דרך בשולחן העבודה ובתפריט התחל.
לעדכונים עתידיים אין צורך לשלוח שוב את כל החבילה: שולחים את קובץ ה-`patch.zip` המתאים, והוא מחליף רק את קוד התוכנה.

לוח הניהול המקומי נמצא ב-`admin.html`. בשלב זה הוא קורא נתונים מ-`localStorage` בלבד,
ולכן הוא מציג רק פעילות ופניות שנשמרו באותו דפדפן. כדי לראות פניות מכל הגולשים צריך לחבר
שירות נתונים מרכזי ב-`site-interactions-config.js`.

כניסה זמנית ללוח הניהול:

- קוד ניהול: `gal-einai`
- סיסמה: `770770`

את פרטי הכניסה מחליפים ב-`admin-config.js` באמצעות ערכי SHA-256 חדשים. באתר סטטי ההגנה הזו
מיועדת להסתרה בסיסית ונוחות עבודה, ולא מחליפה הרשאות שרת אמיתיות.

בלוח הניהול יש אזור העלאת קבצים. במצב הנוכחי הקבצים נשמרים בדפדפן המנהל באמצעות IndexedDB,
ואפשר להוריד או למחוק אותם מתוך הלוח. כדי שהעלאה תעדכן את האתר לכל הגולשים, צריך להגדיר
שרת העלאות ב-`uploadEndpoint` בתוך `admin-config.js`.

בנוסף יש אזור ניהול תוכן וקישורים: הודעות באתר, גרסאות להורדה, צפנים לדוגמה, קישורים שימושיים
והערות מנהל. הפריטים נשמרים בדפדפן המקומי ואפשר לערוך, למחוק ולייצא אותם לקובץ JSON.
