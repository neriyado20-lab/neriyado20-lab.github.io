פיתוח עתידי רציני

גל עיני Web

גרסת הדפדפן תיבנה כפרויקט מסודר ולא כקיצור דרך: מנוע חיפוש מדויק, טקסט תורה מאומת, תצוגת ממצאים יציבה, טבלאות, שמירה ופתיחת צפנים. עד להשלמתה, הגרסה המומלצת להפצה היא תוכנת Windows.

הורד את גרסת Windows ראה צפנים לדוגמה

בסיס חיפוש

בניית מנוע חיפוש לדפדפן עם בדיקות מול תוצאות התוכנה הקיימת, כדי למנוע סטיות בין Windows לבין Web.

תצוגת תורה

תצוגה יציבה סביב הראשית, טבלת ממצאים, מעבר בלחיצה, צבעים וקווים, בלי שממצאים יקפצו או ייעלמו.

שמירה ושיתוף

פתיחת קבצי צופן, שמירה, הדפסה, קישורים לצפנים לדוגמה, ובהמשך אפשרות שיתוף מסודרת.

מצב נוכחי

האתר משמש כרגע להפצה מכובדת של התוכנה, להצגת דוגמאות, ולהכנת תשתית לגרסת Web עתידית. אין כאן התחייבות לגרסת דפדפן מלאה עד שהיא תיבדק כראוי.

התוכנה המלאה זמינה ל-Windows קובץ ZIP להורדה, עם טקסט תורה מובנה ודוגמאות באתר.
